# Global Development Dashboard (GapMinder Data)

This project is an interactive dashboard built with Python, Panel, Plotly, and Pandas to explore global trends in GDP, life expectancy, and population across countries and time.

## Features
- Interactive filtering by region and year
- Scatter plot visualization
- Sankey diagram for grouped comparisons
- Real-time exploration of development indicators

## Tools Used
- Python
- Pandas
- Panel
- Plotly

## How to Run
1. Clone this repository
2. Install required packages
3. Run:

```bash
panel serve gapminder_app.py --show
